## Korner (Żary) — szczegółowo

Pełna referencja Korner: `korner-katalog.md`. Skrót:

- **Płyty meblowe** (Egger, Kronopol, Pfleiderer, Swiss Krono)
- **Fronty** (laminat, lakier, fornir, akryl)
- **Okucia** (Blum, Hettich, GTV)
- **Cięcie + okleinowanie ABS** (CNC ±0.5mm)
- **Brak montażu u klienta** (potrzebny wykonawca lokalny)

**Co Korner robi:** ostateczne formatki gotowe do skręcenia.
**Co Korner nie robi:** instalacja u klienta. Potrzebujesz lokalnego stolarza-montażysty.

## Leroy Merlin / Castorama — wykończenia masowe

Co tam kupić:
- **Farby** (Dulux, Śnieżka, Beckers, Tikkurila — ale lepiej w salonie Tikkurili)
- **Listwy przypodłogowe** (MDF primer 80-100mm)
- **Klej do paneli / spieku** (Sopro, Mapei)
- **Płyty kartonowo-gipsowe + profile + wkręty**
- **Akcesoria narzędziowe** (sznurek murarski, poziomice, taśmy)
- **Podstawowe oświetlenie funkcjonalne** (halogenki LED, taśmy podstawowe)
- **Sanitarne ekonomiczne** (zawory, syfony, akcesoria)
- **Listwy aluminiowe progowe**

Czego NIE kupować tam:
- Mebli do trzymania (kiepska jakość)
- Oświetlenia hero (mało designerskie)
- Drogiej armatury (lepiej u dystrybutora)
- Płytek lasterskich (lepiej u Tubądzina)

## IKEA — kiedy używać, kiedy nie

**Używać:**
- Sofy (Vimle, Söderhamn — sprawdzone, modułowe, łatwo rozbudować)
- Krzesła jadalniane (Lisabo, Henriksdal — solidne, długoletnie)
- Tekstylia (zasłony, dywany, poduszki — sprawdzona jakość)
- Pościel (PUDEROST, BOMSKOTTE — bawełniana, dobra cena)
- Drobne meble do sypialni (komody MALM zostawiamy, alternatywa: HEMNES jeśli premium look)
- Akcesoria łazienkowe (ścierki, dozowniki)
- Storage do garderoby (system PAX wewnątrz, ale fronty lepiej Korner)
- Bazowe oświetlenie (jeśli budżet wymaga)

**Nie używać:**
- Mebli "designerskich" (LANDSKRONA itd. — wyglądają jak imitacja designerskich za 70% mniej, ale jakość 30%)
- Stołów dębowych (klejone elementy, nie litych — Bonami / Westwing lepiej)
- Lamp hero (kompromisowe)

## Allegro — kiedy używać

**Używać dla:**
- Drobnicy elektrycznej (kable, koszulki termokurczliwe, gniazdka modułowe)
- Profile LED + zasilacze (czasem 30% taniej niż Castorama)
- Wkręty, kołki, drobne metalowe akcesoria
- Generic akcesoria łazienkowe (haczyki, mydelnice — nie wymagaj marki)
- Niemarkowy mosiądz dekoracyjny (lampki nocne)

**NIE używać dla:**
- "Designerskich" replik (BHP brak, gwarancji brak)
- Armatury (mogą wycieki, BHP fail)
- AGD (mogą szare = brak gwarancji EU)
- Mebli (transport ciężki, ryzyko, brak zwrotu)

## Negocjacje cenowe — kiedy działają

| Sklep          | Rabat negocjowalny? | Ile zwykle             |
|----------------|----------------------|------------------------|
| **Korner**     | TAK                  | 5-15% przy większym zamówieniu |
| **Leroy / Castorama** | NIE             | nie (chyba że promocja czasowa) |
| **IKEA**       | NIE                  | nie                              |
| **Westwing**   | NIE bezpośrednio     | code zniżkowy w newsletterze     |
| **Bonami**     | NIE                  | sale 2-3× w roku                 |
| **Salony designerskie** | TAK         | 5-25%, im więcej tym lepiej      |
| **Dystrybutorzy AGD** (Onninen, Komfort) | TAK | 10-30% przy całej kuchni AGD |
| **Dystrybutorzy armatury** | TAK      | 15-25% przy całej łazience      |

**Trik:** kup wszystko do łazienki (armatura + ceramika + kabina) u **jednego dystrybutora**, a nie w Castoramie / Leroyu. Cena niższa, gwarancja lepsza, doradca zna produkty.
