# Style i estetyki wnętrz — biblioteka

Referencja stylów wykraczająca poza house style. Stosuj **TYLKO** gdy user wyraźnie poprosi o jeden ze stylów poniżej. Domyślnie używaj **Modern Polish Apartment** (house style).

## Modern Polish Apartment (HOUSE STYLE — default)

**Charakter:** Minimalizm z ciepłym drewnem, dyskretny luksus, soft daylight, czarne akcenty.

| Element       | Specyfikacja                                              |
|---------------|------------------------------------------------------------|
| Paleta        | Orzech royal + kremowy/off-white + czarny + szary kamień  |
| Drewno        | Orzech, ciemny dąb chevron, naturalne wykończenie mat     |
| Tekstylia     | Boucle szary, len natural, wełna kremowa                  |
| Metale        | Czarny mat (uchwyty, armatura, listwy)                    |
| Kamień        | Spiek czarny mat (Dekton Sirius, Neolith Basalt)          |
| Światło       | 3000K LED w wnęce, czarne lampy listwowe                  |
| Atmosfera     | Spokojny, ciepły, dyskretny, bezkonkurencyjny dla relaksu |
| Inspiracje    | Architectural Digest Polska, Dezeen Warsaw apartments      |

**Prompt anchor:** *"modern minimalist Polish apartment with walnut royal smooth matte cabinet fronts, cream interior, thin black bar handles, dark stone matte worktop, warm 3000K LED lighting, soft natural daylight"*

## Japandi

**Charakter:** Japońska prostota + skandynawska funkcjonalność. Spokój, naturalne materiały, asymetria.

| Element       | Specyfikacja                                              |
|---------------|------------------------------------------------------------|
| Paleta        | Off-white + czarny + drewno surowe (dąb / jesion) + zieleń |
| Drewno        | Jasny dąb / jesion, naturalne wykończenie mat lub oil      |
| Tekstylia     | Len natural, bawełna boucle, juta                          |
| Metale        | Czarny mat, czasem mosiądz patynowany                      |
| Kamień        | Travertyn jasny, surowy beton                              |
| Charakterystyka mebli | Niski, low-profile (japoński futon, niskie sofy) |
| Oświetlenie   | Papierowe lampy (Akari / IKEA REGOLIT), miękkie 2700K       |
| Charakter okien | Shoji-inspired (poziome lamele drewniane / żaluzje)      |
| Dekoracje     | Ceramika ręcznie robiona (waza ikebana), pojedyncze rośliny suszone |

**Prompt anchor:** *"japandi interior, light oak natural wood floor, off-white walls, paper pendant lights, low-profile linen sofa, minimal styling, single ikebana arrangement, soft warm daylight, Wallpaper magazine japandi feature"*

**Kluczowe NIE:** żadnego nadmiaru. Mniej znaczy więcej. Asymetria w kompozycji.

## Jak dobrać styl

### Pytania pomocnicze (gdy user nie wie czego chce)

1. **Co czujesz wchodząc do mieszkania marzeń?** → mood = wskazówka stylu
2. **Co cię odrzuca?** → eliminacja stylów (np. "nie znoszę bałaganu" → boho out)
3. **Co masz teraz?** → kierunek ewolucji
4. **Jaki masz aparat / film / Pinterest?** → już masz styl, tylko nie wiesz

### Mixowanie stylów (zaawansowane)

Można łączyć **2 sąsiednie style** (np. japandi + scandi, modern classic + glamour, industrial + mid-century). NIE łącz przeciwstawnych (glamour + industrial = chaos).

**Sąsiedztwa OK:**
- Japandi ↔ Scandi ↔ Mid-Century
- Wabi-Sabi ↔ Japandi
- Modern Classic ↔ Glamour
- Industrial ↔ Mid-Century
- Boho ↔ Coastal ↔ Mediterranean

## Reguła "70-25-5"

W każdym pomieszczeniu — proporcje:
- **70%** głównego stylu (meble, ściany, podłogi)
- **25%** wspierającego stylu lub naturalnych materiałów
- **5%** akcent kontrastowy (jeden hero piece który łamie schemat)

Bez akcentu → nudne. Bez bazy 70% → chaos. Idealna proporcja → wnętrze które wygląda "świadomie zaprojektowane" a nie "kupione na raz".
