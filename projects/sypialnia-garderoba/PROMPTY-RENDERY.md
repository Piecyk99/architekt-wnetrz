# Prompty do generatora fotorealistycznych renderów

Wklej w **Gemini Nano Banana 2** (najlepiej, najwierniej trzyma proporcje wnętrz),
ewentualnie Midjourney v6.1 / DALL-E 3 / Imagen 3.

Każdy prompt skonstruowany w 5-component formuły z prestiżową kotwicą
(Architectural Digest / Dezeen) — zgodnie z house style skilla.

**ZAKAZANE słowa:** "8K", "ultra-realistic", "masterpiece", "best quality" —
degradują output. Zamiast tego ustaw aspect ratio i imageSize: 2K w parametrach.

---

## Konfiguracja (jeśli używasz Gemini API bezpośrednio)

```bash
# Python skrypt (jeśli masz banana skill zainstalowany)
python3 ~/.claude/plugins/cache/banana-claude-marketplace/banana-claude/1.4.1/skills/banana/scripts/generate.py \
  --prompt "<prompt z góry>" \
  --aspect-ratio "16:9" \
  --resolution "2K" \
  --model gemini-2.5-flash-image
```

Lub przez MCP w Claude Code:
```
mcp__nanobanana-mcp__set_aspect_ratio { ratio: "16:9" }
mcp__nanobanana-mcp__gemini_generate_image { prompt: "<full prompt>", imageSize: "2K" }
```

## Iteracja

Jeśli pierwszy render nie pasuje:
- Zmień podłogę → `gemini_edit_image` z tym samym obrazem + "change the floor to lighter natural oak with visible knots"
- Zmień oświetlenie → "make the LED warmer, more amber, evening atmosphere"
- Zmień ramę lustra → "make the mirror frame thicker, 40mm black matt steel"

Tańsze niż regeneracja od zera, zachowuje proporcje pokoju.
